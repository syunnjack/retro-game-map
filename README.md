# retro-game-map

Retro Game Mapは、レトロゲームセンター、懐かし自販機、ゲーム喫茶、閉店情報を検索・確認できるプロトタイプです。

## Concept

- 現存、閉店、要確認の施設ステータスを分けて表示する
- レトロ筐体、格ゲー、音ゲー、自販機、プライズなどで探せる
- 遠征者向けに宿、喫煙所、飲食、最寄駅メモを併記する
- ユーザー投稿と管理者確認で、古い情報やリンク切れを減らす
